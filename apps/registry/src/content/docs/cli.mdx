# CLI Reference

The SkillSpace CLI is your primary tool for interacting with the registry.

## `skillspace init`
Initializes a new `skillspace.yaml` manifest in the current directory.

## `skillspace install <pkg>`
Downloads a package and its dependencies to your local `.skillspace` cache.

## `skillspace run <pkg>`
Executes a capability locally.

## `skillspace publish`
Validates and uploads your capability to the global registry.
