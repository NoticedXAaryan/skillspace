# Getting Started

SkillSpace is the universal package manager and execution runtime for AI capabilities. Think of it like npm, but for LLM prompts, agents, and workflows.

## Prerequisites

- Node.js 18+ installed on your machine
- An API key from one of: OpenAI, Anthropic, or Google

## Installation

Run the CLI directly using `npx`:

```bash
npx skillspace <command>
```

Or install it globally:

```bash
npm install -g skillspace
```

## Configure Your Model

Set up your API key so SkillSpace can call AI models:

```bash
# Option 1: Set environment variable
export OPENAI_API_KEY="sk-..."

# Option 2: Use the CLI to configure
skillspace model add openai
```

## Running Your First Skill

Install a skill from the registry:

```bash
skillspace install @skillspace/code-reviewer
```

Run it against your code:

```bash
skillspace run code-reviewer --input ./src
```

## Searching for Skills

Browse the registry for available skills:

```bash
skillspace search "security"
```

## Publishing Your Own

Create a `skill.yaml` in your project directory, then publish:

```bash
skillspace init
# Edit skill.yaml with your persona definition
skillspace publish
```
